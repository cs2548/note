IsByRef() [v1.1.01+]

如果函数的 ByRef 参数(按地址传递的参数) 与指定的变量一起提供, 则返回一个非零数字.

TrueOrFalse := IsByRef(ParameterVar)

参数

ParameterVar

变量的引用. 例如: IsByRef(MyParameter).

返回值

如果 ParameterVar 是 ByRef 参数(按地址传递的参数) 而且调用者传递了这个参数则返回 1; 否则当 ParameterVar 为其他类型参数时返回 0.

相关

ByRef 参数(按地址传递的参数)

示例

#1

MsgBox, % Function(MyVar)

Function(ByRef Param)
{
    return IsByRef(Param)
}
